# IPL Win Probability Predictor

Predict the winning chances of IPL teams in real time using **Python, FastAPI, and Machine Learning**.  
Enter match details and get live probability predictions for both teams.

## Features
- Predict batting and bowling team win probability
- Real-time calculations of runs left, balls left, wickets left, CRR, and RRR
- Interactive frontend for easy input
- Full-stack project with backend API and ML integration

## Tech Stack
- Python, FastAPI
- Pandas, NumPy
- Scikit-learn (ML model)
- HTML, CSS, JavaScript

## How to Run
1. Create a virtual environment and activate it  
2. Install dependencies: `pip install -r requirements.txt`  
3. Run backend: `uvicorn Backend.main:app --reload`  
4. Open frontend `index.html` in browser
